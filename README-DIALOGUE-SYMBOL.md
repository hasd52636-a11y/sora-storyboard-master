# 对话符号功能使用说明

## 功能概述
对话符号功能允许您在分镜中添加对话气泡，显示15字以内的人物对话内容，并将其自动合成到最终技术提示词中。

## 使用方法

### 1. 添加对话符号
- 点击分镜编辑器中的"添加符号"按钮
- 从符号库中选择"对话气泡"符号
- 将对话气泡拖拽到画布上的合适位置

### 2. 编辑对话内容
- 点击已添加的对话气泡符号
- 在弹出的编辑窗口中，找到"对话内容"输入框
- 输入不超过15字的对话文本（支持中文、阿拉伯语等各种语言）
- 点击"保存"按钮完成编辑

### 3. 对话气泡显示规则
- 对话文本会直接显示在画布上的对话气泡中
- 文本会自动居中并适应气泡大小
- 支持各种语言的文本内容，保持原始输入格式

## 技术提示词生成

对话内容会自动添加到最终的技术提示词中，格式如下：

```
SC-01
   > 内容: 两个人在咖啡馆交谈
   > 空间指令: "对话气泡" @ {左上角}
   > 对话: (Character speaks: "你好，最近怎么样？")
----------------------------------
```

## 注意事项
- 对话内容限制在15字以内，超出部分会被截断
- 对话文本保持原始输入语言，不受技术提示词语言设置影响
- 每个分镜可以添加多个对话符号
