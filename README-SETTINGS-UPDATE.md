# 设置更新说明

## 为什么看不到新的默认设置？

应用会优先从浏览器的 `localStorage` 中加载之前保存的设置，而不是直接使用代码中定义的默认设置。这就是为什么您可能看不到我们对默认API设置的修改。

## 如何看到新的默认设置？

要看到新的默认设置（硅基流动作为默认图片API），请按照以下步骤操作：

1. **打开浏览器控制台**（快捷键：F12 或 Ctrl+Shift+I）
2. **切换到Console标签页**
3. **执行以下命令**：
   ```javascript
   localStorage.removeItem("appSettings");
   ```
4. **刷新页面**

执行后，应用将清除之前保存的设置，并使用新的默认设置。
