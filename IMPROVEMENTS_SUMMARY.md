- 信息密度高

**优化后**:

## 📝 使用指南

### 1. 首次使用

1. 启动应用：
```bash
npm run dev
```

2. 打开浏览器访问 http://localhost:3001

3. 首次启动会自动显示配置向导

4. 输入智谱AI的API密钥（或跳过使用演示模式）

5. 开始创作！

### 2. 获取API密钥

**智谱AI（推荐）**:
1. 访问 https://open.bigmodel.cn
2. 注册并登录
3. 在控制台创建API密钥
4. 复制密钥到配置向导

**其他提供商**:
- 硅基流动: https://siliconflow.cn
- 通义千问: https://dashscope.aliyuncs.com
- Moonshot: https://moonshot.cn

### 3. 智能生成分镜

1. 在主页输入你的故事脚本
2. AI会自动分析并推荐：
   - 最适合的视觉风格
   - 最佳的分镜数量
   - 内容复杂度分析
3. 点击"智能生成分镜"
4. 等待AI生成（通常30秒-2分钟）
5. 查看和编辑结果
6. 导出为图片或PDF

### 4. 高级功能

**手动调整**:
- 点击"高级选项"展开
- 手动选择视觉风格
- 调整分镜数量（2-12个）

**学习用户偏好**:
- 系统会记录你的使用习惯
- 越用越智能
- 自动应用你喜欢的配置
