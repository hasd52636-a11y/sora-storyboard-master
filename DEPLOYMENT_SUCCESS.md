## 🐛 故障排查

### 如果遇到问题

1. **配置向导不显示**
   ```javascript
   // 在浏览器控制台执行
   localStorage.clear()
   location.reload()
   ```

2. **API调用失败**
   - 检查API密钥是否正确
   - 在设置中重新配置
   - 尝试切换API提供商

3. **图片不显示**
   - 检查网络连接
   - 清除浏览器缓存
   - 刷新页面

4. **生成失败**
   - 查看浏览器控制台错误
   - 检查API配额
   - 联系技术支持
