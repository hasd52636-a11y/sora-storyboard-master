## 🎨 完整功能指南

### 1. 分镜编辑

**查看分镜**：
- 左侧显示当前分镜大图
- 右侧显示所有分镜列表
- 点击列表切换分镜

**编辑文本**：
- 点击文本框的"编辑"按钮
- 修改内容
- 点击"确认"保存

**重新生成图片**：
- 悬停在分镜缩略图上
- 点击紫色重新生成按钮
- 自动重新生成该分镜图片

**上传自定义图片**：
- 悬停在分镜缩略图上
- 点击蓝色上传按钮
- 选择图片文件
- ✅ 图片自动显示

### 2. 参考主体功能

**上传参考图片**：
1. 在编辑页面找到"参考主体"区域
2. 点击"上传参考图片"
3. 选择图片文件
4. ✅ 图片显示在参考区域

**一键AI抠图**：
1. 确保已上传参考图片
2. 点击"一键AI抠图"按钮
3. 等待AI处理（10-30秒）
4. ✅ 自动替换为抠图后的透明背景图

**生成三视图**：
1. 确保已上传参考图片
2. 点击"生成三视图"按钮
3. 等待AI生成（20-40秒）
4. ✅ 自动替换为包含三个视图的图片

**使用参考主体**：
- 抠图或生成三视图后
- 重新生成分镜时会参考该主体
- 保持角色/物体一致性

### 3. 符号标注

**添加符号**：
1. 在符号库中选择符号
2. 点击分镜图片放置
3. 拖动调整位置
4. 调整大小和旋转

**编辑符号**：
- 点击符号打开编辑面板
- 修改名称和描述
- 调整位置、大小、旋转
- 保存或删除

### 4. 导出作品

**导出选项**：
- 📄 导出为PDF
- 🖼️ 导出为图片
- 📋 导出为脚本

**操作**：
1. 完成编辑后点击"导出项目"
2. 选择导出格式
3. 下载文件

## 💡 使用技巧

### 技巧1: 快速体验
- 首次使用选择"演示模式"
- 无需配置即可体验所有功能
- 了解流程后再配置API密钥

### 技巧2: 智能推荐
- 输入脚本后查看AI推荐
- 推荐通常是最优配置
- 可以展开"高级选项"手动调整

### 技巧3: 批量生成
- 一次生成所有分镜
- 系统会自动逐个生成图片
- 无需手动操作

### 技巧4: 快速编辑
- 只需修改不满意的部分
- 使用重新生成按钮重新生成单个分镜
- 无需重新生成全部

### 技巧5: 保持一致性
- 使用参考主体功能
- AI抠图去除背景
- 生成三视图了解物体结构
- 所有分镜使用相同主体

## 🔧 常见问题

### Q1: 如何获取API密钥？

**智谱AI**：
1. 访问 https://open.bigmodel.cn
2. 注册并登录
3. 在控制台创建API密钥
4. 复制密钥到配置向导

### Q2: 生成失败怎么办？

**解决方案**：
1. 检查API密钥是否正确
2. 检查网络连接
3. 尝试切换API提供商
4. 查看浏览器控制台错误信息

### Q3: 图片不显示怎么办？

**解决方案**：
1. 确认图片已上传成功
2. 检查文件格式（支持jpg、png、webp）
3. 刷新页面
4. 清除浏览器缓存

### Q4: AI抠图效果不好？

**改进方法**：
1. 使用清晰的参考图片
2. 确保主体明显
3. 尝试多次生成
4. 手动选择主体区域

### Q5: 如何保存我的作品？

**保存方式**：
- 自动保存：系统自动保存到浏览器本地
- 手动导出：点击"导出项目"下载文件
- 云端保存：配置API密钥后自动同步
